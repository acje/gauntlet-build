## Systems not plumbing: Agent first components strategy

- Reduce agent build complexity by sunsetting stale assumptions
- Tailor components for spec driven development by raising abstractions and narrowing scope
- Include only necessary infrastructure; assume other systems can adapt
- Don´t boil the ocean with undifferentiated heavy lifting
