Reusable Rust crates (`quics-*`) for web serving, memoization, and API collection in event-driven architectures.

## Overview

Quicksilver provides a set of composable building blocks for constructing event-driven services in Rust. Each crate is focused on a single responsibility and designed to work independently or together.
