Security-oriented actor model specification for distributed computing with WASM-sandboxed enclaves, capability-based messaging, and supervision hierarchies.

## Overview

The **Actor Enclave Model (AEM)** extends the classical actor model with security-first primitives. Each actor runs within an enclave — a WASM-sandboxed execution environment with explicit capability grants for messaging, state access, and resource usage.
